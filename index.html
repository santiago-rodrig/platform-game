Platform Game

Have you ever played a platform game before? I'm 80% sure you had, but in the chance of me being wrong (20%), all that you have to do is jump, not fall, and gather points (and obviously do not touch the spikes), easy! In this game you'll be running forever, something similar to flappy bird but not that similar, because you are not a bird. Enough talking, go ahead and play!

Controls

Use ↑ to jump, tap the screen if you are using a mobile device.

Mechanics

  1. Jump to avoid falling down (two jumps available when just starting)
  2. Evade the spikes that appear randomly above platforms
  3. Collect jewels that also appear randomly above platforms
  4. Every minute difficulty increases (there are only two more levels of difficulty) and when this happens speed gets augmented, but score and jumps are augmented too!

Game Engine

This game was made with Phaser 3, take a look if you feel like it, but in summary it is a JavaScript library for creating games for the browser using, yes, obviously, JavaScript, and it's completely supported by the developers' community.
